üìñ About Chaturanga

Chaturanga is a comprehensive Android game implementing multiple historical variants of Chaturanga, the ancient Indian predecessor to modern chess. The name ‡§ö‡§§‡•Å‡§∞‡§Ç‡§ó (Chaturanga) means "having four limbs" in Sanskrit, referring to the four divisions of an ancient Indian army.

üéÆ Game Modes

4-Player Dice Mode

Authentic 6th-century gameplay with dice rolls determining which piece type can move each turn.

4-Player Strategic

Same setup as dice mode but without dice - players freely choose their moves for skill-based gameplay.

Persian Shatranj

Historical Persian variant with traditional Shatranj rules and limited piece movements.

Modern Chess

Standard modern chess rules with full implementation of all piece movements.

üîß How to Build

Requirements

  • Android Studio Arctic Fox or later
  • Android SDK 21+ (Android 5.0 Lollipop and above)
  • Java 11+
  • Gradle 7.4.2+

Build Steps

  1. Clone this repository
  2. Open the project in Android Studio
  3. Wait for Gradle sync to complete
  4. Connect an Android device or start an emulator
  5. Click Run (Shift+F10) or use the command:
./gradlew assembleDebug

Testing with Real Ads

The app currently uses Google's test ad unit IDs. To use your own ads:

  1. Create an AdMob account at admob.google.com
  2. Create an app and get your Application ID
  3. Update the Application ID in AndroidManifest.xml
  4. Create ad units and update IDs in Constants.java:
public static final String BANNER_AD_UNIT_ID = "YOUR_BANNER_ID";
public static final String INTERSTITIAL_AD_UNIT_ID = "YOUR_INTERSTITIAL_ID";
public static final String REWARDED_AD_UNIT_ID = "YOUR_REWARDED_ID";
public static final String NATIVE_AD_UNIT_ID = "YOUR_NATIVE_ID";
